# CRITERIOS DE IDENTIDAD VISUAL - PROYECTOS TMC

Esta guía define los estándares visuales obligatorios para la interfaz de usuario.

## 🎨 Emojis Oficiales

| Acción | Icono | Uso Obligatorio |
| :--- | :---: | :--- |
| **Notas** | 📝 | Reemplaza a `📍` o cualquier otro pin. Siempre hoja de papel. |
| **Reprogramar** | 🗓️ | En menús contextuales y acciones de agenda. |
| **Refresh/Carga** | 🔄 | Flecha circular azul. Para botones de actualizar y spinners. |
| **Eliminar (Menús)** | 🗑️ | En menús desplegables o modales de confirmación de borrado. |
| **Eliminar (Tablas)**| ❌ | Acción directa en filas de tablas. |
| **Fabricar** | 🔨 | Acción de Alta de Producción. |
| **Ordenar** | 🔼 / 🔽 | **Minimalista**: Solo aparece en la columna activa. Las demás quedan limpias. |
| **Buscar** | 🔍 | En barras de búsqueda y botones de filtro. |
| **Aviso/Alerta** | ⚠️ | Bloqueos, advertencias o estados "Sin Agendar". |

## 🧹 Reglas de Limpieza

1. **Cero Artefactos**: No debe haber caracteres extraños (`â`, `[]`, `†`) adyacentes a los iconos.
2. **Espaciado**: Los iconos deben tener un espacio simple respecto al texto.
3. **Consistencia**: Un mismo concepto no puede tener dos iconos distintos.
